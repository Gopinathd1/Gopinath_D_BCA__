Stock Price Prediction

This project illustrates the use of the ARIMA (AutoRegressive Integrated Moving Average) model for forecasting stock prices, using a dataset of historical daily prices for ITC Ltd from January 1, 2024, to September 20, 2024. The goal is to predict future stock prices based on historical data. The dataset includes columns for Date, Open, High, Low, Close, and Volume, which serve as the basis for analysis. The ARIMA model combines three components: Auto-regressive (AR), which considers relationships with lagged observations; Integrated (I), which involves differencing to achieve stationarity; and Moving Average (MA), which evaluates the relationship with residual errors. To run the model, users upload the dataset, select ARIMA parameters (p, d, q), and visualize the predicted stock prices.
